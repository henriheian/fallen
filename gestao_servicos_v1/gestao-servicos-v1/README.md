# Gestão de Serviços V1

Este é um sistema simplificado para gestão de ordens de serviço, parceiros e controle financeiro.

## Estrutura do Projeto

- `/backend`: Servidor Node.js com Express.
- `/frontend`: Interface web em HTML/Bootstrap.

## Como Executar

1. Entre na pasta `backend`:
   ```bash
   cd backend
   ```
2. Instale as dependências:
   ```bash
   npm install
   ```
3. Inicie o servidor:
   ```bash
   npm start
   ```
4. Abra o arquivo `frontend/index.html` no seu navegador.

## Funcionalidades

- Cadastro de Serviços
- Gestão de Ordens de Serviço
- Controle Financeiro (Entradas e Saídas)
- Dashboard de Indicadores
